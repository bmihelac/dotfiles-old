h1. Usage

cd into the repo and run this to get the snippets submodule:

<pre><code>
git submodule init
git submodule update
</code></pre>

Install vim files:

<pre><code>
cd ~
ln -s REPOS_DIR/vim ~/.vim
ln -s .vim/myvimrc .vimrc
</code></pre>

h2. Python plugins

rope, rope, ropevim install with easy_install or pip.

h2. OSX specific 

MacVim cloned and installed with:

./configure --with-features=huge \
--enable-rubyinterp \
--enable-pythoninterp \
--with-python-config-dir=/Library/Frameworks/Python.framework/Versions/2.6 \
--enable-cscope

h2. Ubuntu

<pre><code>
sudo apt-get install exuberant-ctags
</code></pre>
